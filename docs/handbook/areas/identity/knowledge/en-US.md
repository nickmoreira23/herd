# Knowledge

Meta-tool unifying 9 first-class blocks into a navigable knowledge base.

## Business

Knowledge is the product's memory. Documents, images, videos, audios, tables, forms, links, feeds, and apps — all live as separate blocks, but Knowledge provides a unified lens for organization (folders), composition (per-block settings), and provenance (origin tracking).

## Product

Accessible at `/admin/knowledge`. Folder-tree UI plus filters by block type. Allows organizing across blocks by context, not by type.

## Architecture

Knowledge is a *meta* tool — no own data. It composes existing blocks. Each referenced block retains ownership; Knowledge aggregates via tags and folder relationships.

## Operations

Adding a block to Knowledge: flag the block manifest for Knowledge participation. Configure composition settings via the tool config.

## Glossary

- **Knowledge folder**: cross-block grouping by context rather than by block type.
- **Origin tracking**: provenance trail per item.
