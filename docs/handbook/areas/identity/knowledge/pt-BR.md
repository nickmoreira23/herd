# Knowledge

Meta-tool que unifica 9 blocks first-class numa base de conhecimento navegável.

## Business

Knowledge é a memória do produto. Documents, images, videos, audios, tables, forms, links, feeds e apps — todos vivem como blocks separados, mas Knowledge dá uma lente unificada de organização (folders), composição (settings por block) e proveniência (origin tracking).

## Product

Acessível em `/admin/knowledge`. UI de folder tree + filtros por tipo de block. Permite organizar transversalmente por contexto, não por tipo.

## Architecture

Knowledge é tool *meta* — não tem dado próprio. Compõe blocks existentes. Cada block referenciado mantém ownership; Knowledge agrega via tags e folder relationships.

## Operations

Adicionar block ao Knowledge: marcar block manifest com flag de Knowledge participation. Configurar settings de composição via tool config.

## Glossary

- **Knowledge folder**: agrupamento transversal por contexto, não por tipo de block.
- **Origin tracking**: trilha de proveniência de cada item.
